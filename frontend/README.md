# Venturi

Sistema de composição e gerenciamento de orçamentos HVAC desenvolvido para engenharia térmica, climatização, ventilação e composição de custos técnicos.

---

## Objetivo

O Venturi tem como objetivo modernizar e estruturar o processo de elaboração de orçamentos HVAC, substituindo fluxos baseados exclusivamente em planilhas por uma plataforma organizada, escalável e orientada à engenharia.

O sistema foi pensado para atender processos de:

- climatização;
- ventilação;
- composição de custos;
- levantamento de materiais;
- mão de obra;
- indiretos;
- versionamento técnico de revisões.

---

## Principais funcionalidades planejadas

### Gestão de Orçamentos

- criação de orçamentos;
- revisão/versionamento (R00, R01, R02...);
- organização por cliente e obra;
- histórico de revisões.

### Estrutura Técnica

Separação por macrogrupos:

- Climatização
- Ventilação
- Indiretos

Categorias técnicas modulares:

- Equipamentos
- Difusão de Ar
- Dutos
- Rede Frigorígena
- Rede Hidráulica
- Elétrica e Automação
- Custos indiretos
- entre outras.

### Composição de Custos

Controle separado de:

- material;
- equipamentos;
- mão de obra;
- indiretos;
- formação de preço / BDI.

### Interface

Interface inspirada em plataformas modernas como:

- ClickUp
- Notion
- Linear

com foco em:

- produtividade;
- clareza visual;
- navegação rápida;
- escalabilidade.
